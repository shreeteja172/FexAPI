# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive experience for everyone.

## Standards

**Expected behavior:**

- Be respectful and constructive
- Give and receive feedback gracefully
- Focus on what's best for the project and community

**Unacceptable behavior:**

- Harassment, trolling, or personal attacks
- Publishing others' private information
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement

Project maintainers will remove, edit, or reject contributions that violate this code of conduct and may ban repeat offenders.

## Reporting

If you experience unacceptable behavior, report it by emailing [smutukundu2006@gmail.com](mailto:smutukundu2006@gmail.com).

All reports will be reviewed and handled confidentially.
